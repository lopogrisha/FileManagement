# Console File Manager (C++17, std::filesystem)

Простий консольний файловий менеджер на C++, який працює в стилі старих файлових оболонок (типу DOS / Norton Commander).  
Керування здійснюється з клавіатури: стрілки, Enter, Backspace, функціональні клавіші (F2–F9).

## Можливості

- Відображення поточного каталогу (шляху)
- Перегляд списку папок та файлів у поточному каталозі
- Навігація по файловій системі:
  - Вхід у папку
  - Перехід до батьківської папки
  - Вибір початкового диска при запуску (C:\, D:\ тощо)
- Операції з файлами та папками:
  - Створення файлу (F2)
  - Створення папки (F7)
  - Перейменування (F3)
  - Копіювання (F4)
  - Переміщення (F5)
  - Видалення (Del) — для файлів і папок (папки видаляються рекурсивно)
- Перегляд текстових файлів у консолі:
  - Підтримуються розширення: `.txt`, `.cpp`, `.h`, `.log`
- Підрахунок розміру файлу або папки (F6)
  - Для папки розмір обчислюється рекурсивно (сума всіх файлів)
- Пошук файлів (F9):
  - За **повною назвою файлу**
  - За **розширенням**

## Використані технології

- C++17
- `std::filesystem` — робота з файлами і каталогами
- `std::vector`, `std::string`
- `_getch()` з `<conio.h>` — зчитування клавіатурних кодів без Enter
- Консольні команди Windows (`system("cls")`)

> ⚠️ Код орієнтований на **Windows** (через використання `<conio.h>` та `cls`).  
> Під Linux / macOS потребуватиме адаптації (інша очистка екрану, інша обробка клавіш).

## Керування

У головному вікні показуються:

- Поточний шлях
- Список папок (з позначкою `<DIR>`)
- Список файлів (із розміром, якщо можливо)
- Один елемент завжди виділений префіксом `>>`

### Клавіші

| Клавіша          | Дія                                           |
|------------------|-----------------------------------------------|
| ↑ / ↓            | Переміщення по списку елементів               |
| Enter            | Якщо вибрана папка — зайти в неї; якщо файл — переглянути (для текстових розширень) |
| Backspace        | Перехід до батьківської папки                 |
| Del              | Видалити вибраний файл або папку (папки — рекурсивно) |
| F2               | Створити новий файл у поточному каталозі      |
| F3               | Перейменувати вибраний файл або папку         |
| F4               | Копіювати вибраний файл/папку в іншу папку    |
| F5               | Перемістити вибраний файл/папку в іншу папку  |
| F6               | Показати розмір вибраного файлу/папки         |
| F7               | Створити нову папку у поточному каталозі      |
| F9               | Пошук файлів (за назвою або за розширенням)   |
| Esc              | Вихід з програми                              |

## Пошук файлів (F9)

Після натискання **F9**:

### 1) Пошук за назвою файлу
Введіть повне ім’я, наприклад:

- `main.cpp`
- `notes.txt`
- `image.png`

Виводяться всі знайдені шляхи (пошук рекурсивний).

### 2) Пошук за розширенням
Введіть розширення:

- `.txt`
- `.cpp`
- `.log`

Показуються всі файли з цим розширенням (рекурсивно від поточного каталогу).
